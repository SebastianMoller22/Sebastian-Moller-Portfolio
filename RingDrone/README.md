## Ring Drone

This project is set to start scientifically. However, the idea is to create a drone with two outer rings that will have a turbine-like blade in them, which will provide lift. The inner blade will be driven from the main body, where it is able to rotate on the roll axis, moving the drone left and right. The outer ring will be driven from the inner ring and will rotate on its relative pitch axis, moving it forward and backward. 

The ideal of this drone design is that the main body will be perfectly telescope stable.
