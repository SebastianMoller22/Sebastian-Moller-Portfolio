## Router Holder

A small project was done to make a router holder that would be placed as the cover for an Ethernet and an ARIEL outlet. This was done so that the round could sit close to the outlet while not needing any additional furniture for it to sit on. 
Several attempts were made to find the exact outlet cover CAD model, but to no avail. So it was manually measured and designed, and with a few attempts, it was able to click into place and have no rattling. Supports were added to make the mount strong. And 3d printing techniques were used to provide the strongest and least wasteful build possible

This was found to be a complete success and is currently in use.
