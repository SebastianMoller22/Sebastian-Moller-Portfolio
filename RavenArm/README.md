## Portable Raven arm Driver
During the months after my graduation *...* contacted me to help with developing a mechanical driver for Raven 2 sergical robot attachemnt. The goal of this project were as follows:
- Drive all actualtor compons simularly to that of the main raven body
- Make it small and portable
- Allow serial comunication to controll it
- have attachment point so it can be used on differnt robotic arms
- Calibrate the encoders
- run all code from an arduino nano
- document setup and assmbally
- connect arm to a OMEGA 7 controller
