## Hydroponics Tower
 This hydroponics project involved creating a plant holder that would distribute an even amount of water to different layers of the system. The idea of this project is to feed water in through the top. It will enter a layer and fill it to a certain point at which the excess then flows to the next layer. Each layer has a main container that houses both the plants and the 100m pipe. A small platform with holes is used to restrict the flow of water. Once the water reaches the bottom, it is collected into the main water storage where the pump is.

From testing, it was found that the dirt used from the plant would sometimes block the holes and result in an overflow. To stop this, a large overflow outlet was added to each layer. 

This project is ongoing and will eventually have a controller that will connect multiple towers together, but only use one pump to feed water
